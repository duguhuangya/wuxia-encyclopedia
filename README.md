# ⚔ 江湖百晓生 — 武侠百科全书

> 一个基于 Vue3 + TypeScript + ECharts 构建的武侠世界交互式百科全书，以金庸《射雕英雄传》为首批数据源。

## ✨ 功能模块

| 模块 | 描述 |
|------|------|
| 🏠 首页 | 水墨暗黑风格首页，飘落树叶动画 |
| 👤 人物志 | 人物卡片 + 五维雷达图 + 对比面板 |
| 🗡️ 兵器谱 | 兵器卡片 + 等级筛选 + 持有者历史 |
| 📖 武功秘籍 | 招式折叠面板 + 克制关系图 |
| 🏯 门派势力 | ECharts 力导向关系拓扑图 + 双层节点 |
| ⏳ 编年史 | 交错时间轴 + 事件筛选 + 统计面板 |
| 🔍 全局搜索 | 全量数据索引 + 智能建议 + 搜索结果面板 |
| ⚙️ 设置 | 双主题切换（水墨暗黑/古风宣纸） |

## 🛠 技术栈

- **Vue 3** + `<script setup>` + TypeScript
- **Vite** 构建
- **Pinia** 状态管理
- **Vue Router** 路由
- **ECharts** 数据可视化
- **VueUse** 组合式工具

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📐 数据结构

所有数据定义在 `src/data/schemas/types.ts`，首批数据源为金庸《射雕英雄传》：
- 9 位人物（郭靖、黄蓉、黄药师、欧阳锋、洪七公、周伯通、杨康、王重阳、段智兴）
- 3 件兵器
- 5 套武功
- 4 个门派
- 6 个时间轴事件
- 8 条人物关系
